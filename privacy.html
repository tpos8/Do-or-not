DO OR NOT 隐私政策

更新日期：2026 年 6 月 30 日

1. 信息收集

DO OR NOT 是轻量选择工具。默认情况下，应用不要求注册账户，也不主动收集姓名、手机号、身份证号或通讯录等个人信息。

2. 本地数据

用户输入的问题和抽签结果如被保存，将仅保存在用户设备本地。卸载应用或清除应用数据后，相关内容可能被删除。

3. 设备权限

应用可能使用设备运动传感器识别摇动动作。该权限仅用于触发抽签交互，不用于识别用户身份或追踪用户位置。

4. 第三方服务

官网使用 GitHub Releases 提供安装包，并读取公开的下载次数。访问 GitHub 时，相关服务可能依据其自身政策处理必要的网络信息。

5. 使用范围

本应用仅适用于轻量生活选择，不应被用于医疗、法律、财务、人身安全等重要决策。

6. 联系方式

如对本政策有疑问，可通过项目 GitHub 仓库提交 Issue。
